About Attendance Hub

Built for reliable and simple student attendance tracking.

Attendance Hub is a student-focused platform for Cebu Eastern College that helps classes log attendance quickly, monitor records clearly, and reduce manual paperwork for both instructors and students.

What the system does

  • Registers student accounts securely.
  • Logs class attendance with course, section, and schedule details.
  • Shows dashboard summaries and per-subject attendance rates.
  • Stores records in a backend JSON database for persistence.

Why this helps

Faculty and students can immediately access updated attendance status without manually compiling spreadsheets. It supports faster follow-up, better visibility, and cleaner student records.
